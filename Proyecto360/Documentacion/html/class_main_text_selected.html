Detailed Description

Clase que gestiona el comportamiento visual del texto principal de un botón en un carrusel, permitiendo subrayar el texto al interactuar con el puntero y controlar si el subrayado es permanente. Implementa las interfaces IPointerEnterHandler y IPointerExitHandler para detectar eventos del puntero.
