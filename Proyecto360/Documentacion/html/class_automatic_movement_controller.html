Detailed Description

Controlador de movimiento automático para la cámara panorámica cuando el usuario está inactivo. Realiza un recorrido completo de 360° a una velocidad personalizable. Se detiene inmediatamente ante cualquier interacción del usuario. MODIFICADO: Inicia desactivado por defecto.
